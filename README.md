# Calculadora Cyberpunk

Aplicación de escritorio desarrollada en Java utilizando JavaFX. Permite realizar operaciones matemáticas básicas y avanzadas mediante una interfaz gráfica estructurada e historial de operaciones.

---

## ¿Qué hace esta aplicación?

La calculadora permite realizar diferentes tipos de operaciones:

* Suma
* Resta
* Multiplicación
* División
* Potencia
* Raíz

La lógica está organizada en un modelo basado en clases para cada operación, facilitando la escalabilidad del proyecto.

---

## Requisitos

Antes de ejecutar el proyecto, necesitas:

* Java JDK 17 o superior
* Maven

---

## Instalación

### 1. Clonar el repositorio

```bash
git clone https://github.com/tu-usuario/calculadora-java.git
cd calculadora-java
```

---

## Ejecución en Linux

### Compilar el proyecto

```bash
mvn clean install
```

### Ejecutar

```bash
mvn javafx:run
```

---

## Ejecución en Windows

## Uso

1. Inicia la aplicación
2. Ingresa los valores numéricos
3. Selecciona la operación
4. Visualiza el resultado en la interfaz
